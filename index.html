Inicio

Este proyecto presenta una solución eficiente para la gestión de datos médicos utilizando Python. Incluye estructuras avanzadas como árboles binarios, colas de prioridad y grafos, aplicadas a casos reales como la gestión de pacientes y optimización de rutas de ambulancias.

Estructuras de Datos

  • Árboles Binarios de Búsqueda para organizar pacientes.
  • Colas de Prioridad para gestionar urgencias.
  • Grafos para modelar redes de hospitales y rutas.
